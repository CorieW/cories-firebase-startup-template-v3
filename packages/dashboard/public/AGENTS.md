# packages/dashboard/public

Static assets served directly by the dashboard app.

## Directories

- None.

## Files

- `favicon.ico`: Browser favicon asset.
- `logo192.png`: PWA or app icon asset at 192px.
- `logo512.png`: PWA or app icon asset at 512px.
- `manifest.json`: Web app manifest.
- `robots.txt`: Search crawler directives.

## Writing Rules

- Keep asset filenames stable when browser manifests or HTML reference them directly.
- Document externally sourced or regenerated assets when they replace existing files.
