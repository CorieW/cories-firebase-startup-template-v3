# packages

Monorepo workspace hub for backend, common, dashboard, admin, marketing, and docs packages.

## Directories

- `admin/`: Internal TanStack Start admin workspace.
- `back/`: Firebase Fns backend workspace.
- `common/`: Shared types, utils, and repo-contract tests.
- `dashboard/`: TanStack Start dashboard workspace.
- `docs/`: Public TanStack Start docs workspace powered by MDX content.
- `marketing/`: Static React marketing site workspace.

## Files

- None.

## Writing Rules

- Treat each child workspace as separate documented package.
- Update index when packages are added, removed, or repurposed in big way.
